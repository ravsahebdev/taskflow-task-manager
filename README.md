# ⚡ TaskFlow – Task Manager

A modern and responsive **Task Manager** built using **HTML, CSS, and Vanilla JavaScript**. TaskFlow helps users organize daily tasks with a clean interface, category-based management, theme switching, and local storage support.

---

## 🚀 Features

* ✅ Add New Tasks
* ✏️ Edit Existing Tasks
* 🗑️ Delete Individual Tasks
* 🧹 Clear All Tasks
* ✅ Mark Tasks as Complete / Undo
* 🔍 Live Search
* 📂 Category Filters (Work, Personal, Study)
* 🌙 Dark Mode
* ☀️ Light Mode
* 💾 Local Storage Persistence
* 📊 Live Task Counters
* 📱 Fully Responsive Design
* ⚡ Fast & Lightweight
* 🎨 Modern UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* Local Storage API
* DOM Manipulation
